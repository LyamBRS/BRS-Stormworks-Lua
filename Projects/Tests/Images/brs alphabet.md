# What is this?
This is all BRS alphabet as base64 images. Each are 5x5
They were generated from the `drawImage.lua` by doing rectangles.

## The alphabet
### A
#### BRIO BASE64
```
    AAA@A:AAFBACFBADBCEDBC
```
#### Code used in `drawImage.lua`
```lua
    setColor(0,0,0)
    drawRectF(0,0,5,1)
    drawRectF(0,2,5,1)
    drawRectF(0,3,1,2)
    drawRectF(4,3,1,2)
```
